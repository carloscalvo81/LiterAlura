📝 Sobre o Projeto

O Literalura é uma aplicação desenvolvida em Java com Spring Boot, criada para consumir dados da API Gutendex e permitir a exploração de informações literárias através de consultas inteligentes.

A aplicação possibilita:

Buscar livros na API

Armazenar informações no banco de dados

Realizar consultas e estatísticas sobre os dados

O projeto foi desenvolvido com foco em aprendizado e prática dos conceitos abordados no programa Oracle Next Education (ONE), aplicando boas práticas de desenvolvimento com Spring Boot, JPA e PostgreSQL.

🚀 Funcionalidades Principais
🔍 Buscar livros por título

Consulta à API Gutendex e persistência automática no banco de dados.

🗂️ Listar livros e autores

Visualização completa de todos os registros armazenados.

🕵️ Buscar autores vivos em determinado ano

Consulta autores considerando suas datas de nascimento e morte.

🌐 Listar livros por idioma


Suporte para os idiomas:

Português (PT)

Inglês (EN)

Espanhol (ES)

Francês (FR)


👤 Buscar autor por nome

Consulta eficiente diretamente no banco de dados.

🏆 Top 10 livros mais baixados

Ranking com os livros mais populares.


Tecnologias

Java 21

Spring Boot 3.x

Spring Data JPA

PostgreSQL 15+
